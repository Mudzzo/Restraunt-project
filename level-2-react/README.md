# Restaurant Project

## How to Run the Project

### 1. Clone the Repository

```bash
git clone <repository-link>
cd <project-folder>
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

```bash
npm start
```

This will:

- Compile your Sass files into CSS.

### 4. Build the CSS (Optional)

```bash
npm run build:css
```
